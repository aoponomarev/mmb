# pc-home (Домашний ПК)

Эталонное окружение до появления проблем с Node и рассинхроном при открытии проекта на офисном ПК.

| Компонент | Значение |
|-----------|----------|
| **Node** | 18.x (путь/версия могли отличаться от офиса; nvm или прямая установка) |
| **Python** | Присутствовал для node-gyp; точная версия при необходимости уточнить |
| **Build** | Достаточно для сборки better-sqlite3 (VS Build Tools или аналог) |
| **Примечание** | Окружение было рабочим; использовать как референс «known good» при настройке новой машины |

При добавлении нового ПК или восстановлении окружения сверяться с id:env-pc-readme (.cursor/environment-pc/README.md).
