# Node 18 for this project

`better-sqlite3` is compiled for Node 18 (NODE_MODULE_VERSION 127). Use Node 18 when running preflight and MCP.

Project is configured to use **C:\Program Files\nodejs**. Ensure Node 18 LTS is installed there. New terminals in this workspace get that path via `.vscode/settings.json`.

One-time after install: `npm rebuild better-sqlite3` then `npm run preflight`.
