# Докер и Контейнеризация (Docker / Control Plane)

> Статус: Отложено (В бэклоге)
> Приоритет: Самый низкий (Отдаленная перспектива)
> Контекст: Перенос Control Plane и Docker-конфигураций из старой архитектуры.

## Описание
Временно откладываем миграцию Docker-контуров и скриптов `docker-compose`. Сначала необходимо выстроить фундамент бизнес-логики, SSOT путей, политику управления секретами и общие контракты приложения.
Когда приложение локально станет стабильным и предсказуемым, мы вернемся к этой задаче, чтобы обернуть его в изолированные контейнеры.
