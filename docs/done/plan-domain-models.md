## Решение

Перенос математических/финансовых доменных моделей отложен до появления первого файла с математическим кодом в Target App. На текущем этапе нет кода, к которому эти модели относились бы.

## Условие активации

- [ ] Первый файл с математическим кодом перенесён в Target App
- [ ] Causality-факторы `GM.*`/`CM.*` активированы (если полный реестр будет внедрён)
- [ ] `arch-math-models.md` создан со статусом `active`
