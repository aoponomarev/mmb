# Protocol: Command `ОМК` (Brevity)

## Command
`ОМК` = отвечай максимально кратко без потери технической точности.

## Behavior
1. Essence only: command, value, path, result.
2. No fluff or decorative phrasing.
3. Keep precision and actionable details.

## Combination
- `ОМК` + `ЕИП`: short SSOT/path consistency report.
